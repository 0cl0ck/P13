## Objet du document
Ce document a pour objectif de **modéliser l’architecture** qui sera réalisée par les équipes de développement.  
Il inclut :  
- La modélisation graphique de l’architecture  
- Les **principes d’architecture** retenus  
- La **justification** des choix architecturaux  
- Les éventuelles **étapes de transition** nécessaires pour arriver à l’architecture cible
